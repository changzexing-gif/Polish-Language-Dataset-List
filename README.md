# Polish Language Datasets Collection

## Introduction

This repository contains a curated metadata collection of **40 publicly available Polish language datasets**.

It is designed to help NLP researchers, linguists, and students quickly locate resources for various tasks such as **Sentiment Analysis**, **Emotion Recognition**, **Hate Speech Detection**, and **Historical Linguistics** research.

**Key Metadata Fields:**
- **Dataset Name**: The official name of the resource.
- **Validate Dataset URL**: Valid dataset address link.
- **Modality**: The type of data (e.g., Text).
- **Tasks**: Applicable NLP tasks.
- **Original Paper URL**: Link to the original paper where the dataset was published.
- **Description**: A brief summary of the dataset's content and purpose.
